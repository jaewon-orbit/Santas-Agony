# 🎅 Santa's Agony: Optimization with Algorithms
This project was developed as a team project for a university algorithms course (2021)
[View Proposal PDF](./Santa's%20Agony_%20Proposal.pdf)


This holiday, good children receive gifts from Santa 🎁  
To make this possible, we developed an optimization system using:

- 🎯 Knapsack Algorithm (gift selection)
- 🗺️ Traveling Salesman Problem (route optimization)

---

## 📌 Project Overview

Santa faces two main challenges:

1. **Choosing the best gifts within capacity**
2. **Delivering them efficiently to children**

We solve these using classic algorithms:

| Problem | Algorithm |
|--------|----------|
| Gift Selection | Knapsack (Greedy / DP) |
| Delivery Route | Traveling Salesman Problem (Dynamic Programming) |

## 🎁 Knapsack (Gift Selection)

- Input: price, quantity, weight  
- Value: worth = price × quantity / weight  
- Goal: maximize total worth within capacity
